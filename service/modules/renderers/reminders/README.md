# 提醒事项

主要用于长期的提醒事项备忘📝；例如：生日、纪念日。

## 格式

```reminder
# 标题
title: 纪念日！
# 详细的描述
description: 我们恋爱了！
# 开始日期
dates:
  start: 2014-12-24
# 计算天数时是否把“开始日期”包含在内
exclusive: false
# “提醒”设置
remind:
  # 在第几天提醒？
  days: [100,520,1314]
  # 在第几个月提醒？
  months: [1]
  # 在第几年提醒？
  years: [1,10,20,50,100]
```
